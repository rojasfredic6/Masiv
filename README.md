# Masiv - Tecnical Test
#### Prueba tecnica FrontEnd desarrollada con Javascript, Vue, Vuex, Stylus. 

## Clonar repositorio
```
git clone https://github.com/rojasfredic6/Masiv.git
cd Masiv
```
## Instalar dependencias
```
yarn install
npm run install
```

## Compilar e iniciar servidor local
```
yarn serve
npm run serve
```

## Compilar para produccion
```
yarn build
npm run build
```

## Para Indentacion
```
yarn lint
npm run lint
```
